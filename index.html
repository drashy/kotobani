    SOME BACKGROUND, THE BUG AND FURTHER DEVELOPMENT:

    I wrote this game for a couple of reasons.
    First of all I was in a one and half months job hiatus, with plenty of time to spare. I also wanted to check out this newfangled Godot Engine which kept creeping up in the tech news I read. Additionally, I wanted to create a Bookworm clone with better localization, as the original, whilst a lot of fun, was lacking when playing in anything but english.
    My six weeks or so of freedom from pressure are now over, and sadly indentured rent-paying work has a way of crushing to death any creativity I might have in my free time, so I will probably stop working on Kotoboani altogether and never fix The Bug, but the game is open source so maybe someone will.
    The Bug is simple : sometimes (And that's the main problem), tiles in the second row will not fill up when a word has been found split between the first and the second row. As I said : if you find out why, you're welcome to fix it yourself or tell me what you found.

    Another thing : the Godot Engine is actually awesome. At least for this kind of games. The 3D capacities seem to be very nice too, but having no time to check them I prefer to abstain from any judgement. If you want to create games easily without investing big $$$$ (Read : "without buying a Unity3D license") you should definitely check it out. Thanks a lot to the Godot Devs for the engine and the support in their IRC channel. so yeah .. sorry for the bug, I hope you enjoy the game anyway.

    HOW TO BUILD:

    You will need the Godot Engine. Add the Kotobani source to the engine's project list and either run it inside the editor or export it to your platform of choice.

    RULES:

    Form 3 letter words using adjacent tiles. the tiles must touch, but it doesn't matter where they touch. Diagonals are allowed. And you don't have to continue in the same direction to finish the word. So, if you can find an A with an R left from it, then another M ABOVE the previous R you would have a word. CLick twice on the ending letter to validate. Right mouse click restart the word. P to pause.
    Rainbow tiles multiplies the score of the final words. Each rainbow tile increase the multiplicator by 1, starting at 2. So a 3 letter word would bring 3 points, if it uses a rainbow tile, you will get 6 points (3x2), if you can get 2 rainbow tiles in it, you will get 3x3 points, etc ... Note that the longer the word the more points a tile brings anyway.
