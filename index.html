    • Kotobani

    A simple word puzzle game writen with the Godot Engine.

    Project maintained by sammyf Hosted on GitHub Pages — Theme by mattgraham

    LICENSE:

    GPLv3

    BUILD:

    You will need the Godot Engine. Add the Kotobani source to the engine's project list and either run it inside the editor or export it to your platform of choice.

    BINARIES:

    • Linux 64bit
    • Win 64bit
    • Win 32bit
    • OSX 64bit
    Note : the OSX and Win 32bit binaries have not been tested, as I lack the hardware to do so.

    RULES:

    Form 3 letter words using adjacent tiles. the tiles must touch, but it doesn't matter where they touch. Diagonals are allowed. And you don't have to continue in the same direction to finish the word. So, if you can find an A with an R left from it, then another M ABOVE the previous R you would have a word. CLick twice on the ending letter to validate. Right mouse click restart the word. P to pause.

    The music, sound effects and the flying yoga frog are from the wonderful assets from the game Glitch and licensed under Creative Common. You can find out more at http://www.glitch.com
